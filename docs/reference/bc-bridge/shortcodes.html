BC Bridge Shortcodes Reference

Complete reference of all shortcodes available in BC Bridge for displaying products, cart, and checkout functionality.

[bc_products]

Display a grid of products.

Attributes

Attribute Default Description
limit 12 Number of products to display
columns 4 Grid columns (2, 3, or 4)
category - Filter by category slug
orderby date Sort by: date, price, title
order desc Sort order: asc, desc

Examples

// Basic product grid
[bc_products]

// 8 products, 3 columns
[bc_products limit="8" columns="3"]

// Products from specific category
[bc_products category="clothing" limit="12"]

// Sorted by price, ascending
[bc_products orderby="price" order="asc"]

[bc_product]

Display a single product.

Attributes

Attribute Default Description
id - BigCommerce product ID
slug - Product URL slug

Examples

// By product ID
[bc_product id="123"]

// By product slug
[bc_product slug="awesome-product"]

[bc_cart]

Display the shopping cart. No attributes.

Example

[bc_cart]

[bc_checkout]

Display checkout. Renders embedded checkout or redirects based on settings.

Attributes

Attribute Default Description
type auto Checkout type: auto, embedded, redirect

Example

[bc_checkout]

// Force embedded checkout
[bc_checkout type="embedded"]

// Force redirect checkout
[bc_checkout type="redirect"]

Customizing Shortcode Output

Using Filters

// Modify shortcode output
add_filter('bc_bridge_shortcode_output', function($output, $shortcode, $atts) {
    if ($shortcode === 'bc_products') {
        // Customize output
        $output = '<div class="custom-products">' . $output . '</div>';
    }
    return $output;
}, 10, 3);

Creating Custom Shortcodes

// In your theme's functions.php
function my_custom_product_shortcode($atts) {
    $atts = shortcode_atts([
        'id' => 0,
        'featured' => false,
    ], $atts);
    
    if ($atts['featured']) {
        $products = bc_bridge()->api->get_products(['featured' => true]);
    } else {
        $product = bc_bridge()->api->get_product($atts['id']);
    }
    
    ob_start();
    // Render your custom template
    include get_template_directory() . '/bc-bridge/custom-product.php';
    return ob_get_clean();
}
add_shortcode('my_custom_product', 'my_custom_product_shortcode');
