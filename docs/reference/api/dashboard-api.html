Dashboard API Reference

Complete reference for all API routes available in the migration dashboard.

Base URL

All API routes are prefixed with /api:

https://your-dashboard.com/api

Assessment Endpoints

Assess Products

POST /api/assess/products

Analyze product catalog for migration readiness.

Assess Categories

POST /api/assess/categories

Analyze category structure.

Assess Customers

POST /api/assess/customers

Analyze customer data.

Assess Orders

POST /api/assess/orders

Analyze order history.

Assess SEO

POST /api/assess/seo

Analyze SEO elements.

Assess Plugins

POST /api/assess/plugins

Analyze installed plugins.

Assess Custom Data

POST /api/assess/customData

Analyze custom data structures.

Migration Endpoints

Setup Categories

POST /api/migrate/setup-categories

Migrate categories (Phase 1: Foundation).

Migrate Category Products

POST /api/migrate/category

Migrate products for a specific category.

Migrate Customers

POST /api/migrate/customers

Migrate customer accounts.

Migrate Orders

POST /api/migrate/orders

Migrate order history.

Migrate Coupons

POST /api/migrate/coupons

Migrate discount codes.

Migrate Reviews

POST /api/migrate/reviews

Migrate product reviews.

Migrate Pages

POST /api/migrate/pages

Migrate CMS pages.

Migrate Blog

POST /api/migrate/blog

Migrate blog posts.

Connection Endpoints

Connect WooCommerce

POST /api/connect

Connect to WooCommerce store.

Connect BigCommerce

POST /api/bc/connect

Connect to BigCommerce store.

Validation Endpoints

Validate Migration

POST /api/validate

Validate migrated data.

Detailed Validation

POST /api/validate/detailed

Perform detailed validation with sampling.

Export Endpoints

Export Products

POST /api/export/products

Export product data.

Export Customers

POST /api/export/customers

Export customer data.

Export Migration Spec

POST /api/export/migration-spec

Export migration specification.

Export Redirects

POST /api/export/redirects

Export URL redirect mappings.

Preview Endpoints

Preview Products

POST /api/preview/products

Preview product display.

Preview Live Store

POST /api/preview/live

Preview live storefront.

Verification Endpoints

Verify SSL

POST /api/verify/ssl

Verify SSL certificate.

Verify Payment

POST /api/verify/payment

Verify payment gateway configuration.

Verify Shipping

POST /api/verify/shipping

Verify shipping configuration.

Verify Tax

POST /api/verify/tax

Verify tax settings.

Verify Analytics

POST /api/verify/analytics

Verify analytics setup.
