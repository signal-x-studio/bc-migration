  • BigCommerce API - BigCommerce REST API endpoints

API Overview

WooCommerce REST API

Used for reading data from source stores:

  • Products and variants
  • Categories
  • Customers
  • Orders
  • Plugins and settings

BigCommerce REST API

Used for writing data to target stores:

  • Catalog API (products, categories)
  • Customers API
  • Orders API
  • Storefront API (for BC Bridge plugin)

Rate Limits

  • BigCommerce: 150 requests per 30 seconds
  • WooCommerce: No hard limit (paginated requests)
