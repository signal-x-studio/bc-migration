6.2 Go/No-Go Decision Framework

GO to Phase 1 requires ALL of:

  • WPEngine commits to formal partnership (written)
  • 3+ pilots complete successfully (production live)
  • Pilot NPS > 40 (surveyed)
  • SEO impact < 10% loss (measured)
  • Technical integration stable (no critical bugs)
  • Budget on track ( 10%)

NO-GO Automatic if ANY of:

  • WPEngine explicitly declines
  • Fundamental technical barrier discovered
  • Migration timeline > 150% of Shopify benchmark
  • SEO impact > 20% loss
  • Pilot merchants refuse to launch
