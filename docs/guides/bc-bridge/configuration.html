BC Bridge Configuration Guide

Complete guide to configuring the BC Bridge WordPress plugin.

Accessing Settings

Go to WordPress Admin → Settings → BC Bridge

Or use the plugin menu: Plugins → BC Bridge → Settings

API Configuration

BigCommerce API Credentials

Store Hash

  • Your BigCommerce store hash (alphanumeric, no dashes)
  • Found in BigCommerce Admin → Settings → API Accounts

Client ID

  • Your BigCommerce API client ID
  • Created when setting up API account

Access Token

  • Your BigCommerce API access token
  • Keep this secure (stored encrypted)

Test Connection

  • Click "Test Connection" to verify credentials
  • Should show "Connected to [Store Name]"

Display Settings

Products Per Page

Set how many products to show per page:

  • Default: 12
  • Options: 12, 24, 48, 96

Default Sort

Set default product sorting:

  • Name (A-Z)
  • Name (Z-A)
  • Price (Low to High)
  • Price (High to Low)
  • Date Added (Newest)
  • Date Added (Oldest)

Show Stock Status

Toggle stock status display:

  • Show stock badges
  • Hide stock status

Checkout Settings

Checkout Type

Choose checkout method:

Embedded Checkout (Recommended)

  • Checkout in iframe on your WordPress site
  • Customers stay on your domain
  • Requires domain matching

Redirect Checkout

  • Full redirect to BigCommerce checkout
  • Customers leave your site
  • Works with any domain

Thank You Page

Select WordPress page for order confirmation:

  • Choose from your pages
  • Or use default BigCommerce page

Cache Settings

Product Cache TTL

How long to cache product data:

  • Default: 5 minutes
  • Increase for better performance
  • Decrease for real-time updates

Category Cache TTL

How long to cache category data:

  • Default: 15 minutes
  • Categories change less frequently

Clear Cache

Manually clear all caches:

  • Click "Clear All Caches"
  • Use after bulk product updates

Route Configuration

Configure which routes BC Bridge handles:

Product Routes

Shop Page

  • Route: /shop
  • Intercepts WooCommerce shop page
  • Shows BigCommerce products

Product Detail

  • Route: /product/*
  • Intercepts product pages
  • Shows BigCommerce product details

Category Pages

  • Route: /product-category/*
  • Intercepts category pages
  • Shows BigCommerce category products

Cart & Checkout

Cart Page

  • Route: /cart
  • Intercepts cart page
  • Shows BigCommerce cart

Checkout Page

  • Route: /checkout
  • Intercepts checkout page
  • Shows BigCommerce checkout

Advanced Settings

Debug Mode

Enable verbose logging:

  • Enable for troubleshooting
  • Disable for production

Log Location:

  • wp-content/debug.log (if WP_DEBUG enabled)
  • Or check plugin logs

API Rate Limiting

Configure API request handling:

  • Automatic rate limiting enabled
  • Queue requests if rate limit exceeded
  • Retry with exponential backoff

Custom CSS

Add custom CSS for styling:

  • Override default styles
  • Theme integration
  • Custom product card styles

Theme Compatibility

BC Bridge works with most WordPress themes:

Tested Themes

  • Twenty Twenty-Four
  • Astra
  • GeneratePress
  • Storefront
  • Kadence

Custom Themes

For custom themes:

  • BC Bridge provides default templates
  • Override templates in theme
  • Use WordPress hooks for customization

Performance Optimization

Caching

Enable caching for better performance:

  • Use WordPress caching plugins
  • Configure cache TTL settings
  • Clear cache after updates

CDN

BC Bridge uses BigCommerce CDN for images:

  • Images served from BC CDN
  • No need to download to WordPress
  • Fast image loading

API Optimization

Reduce API calls:

  • Increase cache TTL
  • Use batch requests
  • Enable request queuing

Security Settings

API Credentials

  • Credentials stored encrypted
  • Never expose in frontend
  • Use secure storage

HTTPS

  • Require HTTPS for API calls
  • Secure cookie transmission
  • SSL certificate required

Troubleshooting

Settings Not Saving

Check:

  • User permissions (admin required)
  • WordPress nonce validation
  • PHP error logs

Cache Not Clearing

Check:

  • Cache plugin compatibility
  • Transient cache cleared
  • Browser cache cleared

Performance Issues

Check:

  • Cache TTL settings
  • API response times
  • Server resources
