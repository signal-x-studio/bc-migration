What is BC Bridge?

BC Bridge is a WordPress plugin that enables your WordPress site to use BigCommerce as the commerce backend while keeping your WordPress frontend, theme, and content.

Key Features

  • Product catalog display from BigCommerce
  • Shopping cart functionality
  • Checkout flow (embedded or redirect)
  • Customer account integration
  • WooCommerce route interception
