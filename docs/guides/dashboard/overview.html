Dashboard Features Overview

The migration dashboard provides a comprehensive interface for managing your WooCommerce to BigCommerce migration. This guide covers all major features and pages available in the dashboard.

Getting Started

Setup Page

Location: /setup

The setup page is the first page you'll see. It guides you through:

  • Connecting to your WooCommerce store
  • Connecting to your BigCommerce store
  • Verifying API credentials
  • Initial configuration

Home Page

Location: /

The home page provides:

  • Migration progress overview
  • Quick access to key features
  • Path recommendations based on your store
  • Next steps guidance

Assessment Features

The dashboard includes comprehensive assessment tools to analyze your WooCommerce store before migration.

Products Assessment

Location: /products

Analyzes your product catalog:

  • Total product count
  • Product types (simple, variable, grouped, external)
  • Variation count
  • Image coverage
  • SKU coverage
  • Category assignments
  • Complexity indicators

Categories Assessment

Location: /categories

Analyzes your category structure:

  • Total category count
  • Category hierarchy depth
  • Category image coverage
  • Product distribution
  • Naming consistency

Customers Assessment

Location: /customers

Analyzes your customer data:

  • Total customer count
  • Customer address coverage
  • Email validation
  • Customer metadata

Orders Assessment

Location: /orders

Analyzes your order history:

  • Total order count
  • Order status distribution
  • Order date range
  • Order value statistics

SEO Assessment

Location: /seo

Analyzes SEO elements:

  • Meta title coverage
  • Meta description coverage
  • URL structure
  • Image alt text
  • Schema markup

Plugins Assessment

Location: /plugins

Analyzes installed plugins:

  • Active plugin list
  • WooCommerce extensions
  • Compatibility assessment
  • Migration blockers
  • Alternative recommendations

Custom Data Assessment

Location: /custom-data

Analyzes custom data:

  • Custom post types
  • Custom fields
  • Custom taxonomies
  • Third-party data

Migration Features

Migration Wizard

Location: /migrate

The migration wizard guides you through the migration process:

  • Phase 1: Foundation - Category migration
  • Phase 2: Core Data - Products and customers
  • Phase 3: Transactions - Orders and coupons
  • Phase 4: Content - Reviews, pages, and blog

Features:

  • Real-time progress tracking
  • Phase-by-phase migration
  • Error handling and reporting
  • ID mapping preservation
  • Resume capability

Export Features

Location: /export

Export data for external use:

  • Product export (CSV, JSON)
  • Customer export (CSV, JSON)
  • Migration specification export
  • Redirect mapping export

Path Selection

Paths Overview

Location: /paths

Choose your migration path:

  • BC Bridge - WordPress plugin integration
  • Headless - Custom frontend with BigCommerce backend
  • Stencil - BigCommerce Stencil theme
  • Makeswift - Makeswift visual builder

Path-Specific Guides

  • /paths/bc-bridge - BC Bridge path details
  • /paths/headless - Headless path details
  • /paths/stencil - Stencil path details
  • /paths/makeswift - Makeswift path details
