Getting Started for Stakeholders

Business context, strategic overview, and ROI for the BC Migration initiative.

Executive Summary

BC Migration enables WordPress site owners to upgrade from WooCommerce to BigCommerce as their commerce backend without rebuilding their website. This addresses a key market opportunity: 8,000+ mid-market WooCommerce stores at WP Engine that represent untapped migration targets.

The Opportunity

Market Context

  • WP Engine Partnership: Real partnership opportunity with 8,000+ WooCommerce customers
  • Mid-Market Focus: Stores with $2-5M annual GMV, hitting WooCommerce scaling limits
  • Low-Friction Migration: Unlike traditional replatforming, WordPress stays intact

Business Value

For Merchants:

  • Upgrade commerce backend without losing WordPress investment
  • Better scalability and reliability than WooCommerce
  • Reduced maintenance burden
  • Maintain SEO rankings and content

For BigCommerce:

  • Access to large, qualified customer base via WP Engine partnership
  • Lower-friction migration path increases conversion
  • Competitive advantage vs. "rip and replace" alternatives

Partnership Context

WP Engine Partnership

  • Status: Real partnership opportunity (CFO/CEO level discussions)
  • Customer Base: 8,000+ mid-market WooCommerce stores
  • Pilot Customers: 3 customers + 3 agencies already lined up
  • Value Exchange: WP Engine provides distribution, BigCommerce provides migration tooling

Competitive Advantage

  • vs. Shopify: Lower migration friction (keep WordPress)
  • vs. Traditional Replatforming: No "rip and replace" required
  • vs. WooCommerce: Better scalability and reliability

Risks & Mitigations

Technical Risks

  • WooCommerce Plugin Complexity: Mitigated by assessment engine and plugin mapping
  • API Rate Limits: Handled with rate limiting and batching
  • Data Transformation: Comprehensive mapping and validation

Business Risks

  • Partnership Terms: Validated at CFO level
  • Market Adoption: Pilot customers already committed
  • Competitive Response: First-mover advantage in WordPress headless commerce

Success Criteria

Phase 0 Go/No-Go

GO if ALL are true:

  • WP Engine commits to formal partnership
  • 3+ pilot migrations complete successfully
  • Pilot NPS > 40
  • SEO impact < 10% traffic loss
  • Technical integration stable

NO-GO if ANY are true:

  • WP Engine explicitly declines
  • Fundamental technical barrier
  • Migration timeline > 150% of Shopify
  • SEO impact > 20% loss
