Payment Gateway Plugins

Payment gateway migrations are typically the simplest. BigCommerce includes native integrations for all major payment processors.

WooCommerce Plugin BigCommerce Solution Type Migration Notes
WooCommerce Stripe Gateway Native Stripe Integration Native Control Panel config via API keys. No plugin required.
WooCommerce PayPal Payments Native PayPal Integration Native Core integration eliminates security update concerns.
WooCommerce Square Native Square Integration Native Seamless POS and online synchronization.
WooCommerce Authorize.Net Native Authorize.Net Integration Native Pre-integrated gateway in payment settings.

WordPress-Only Plugins (Keep in WordPress)

These plugins are not commerce-related and remain unchanged in WordPress after migration. This is a key advantage of the "Keep WordPress" architecture.

Plugin Category Example Plugins Type Migration Notes
SEO Yoast SEO, RankMath, All in One SEO Keep in WP Continues managing WordPress content SEO. Product SEO handled by BigCommerce native fields.
Page Builders Elementor, Divi, WPBakery Keep in WP No migration needed. Continue using for all non-product pages.
Forms Gravity Forms, WPForms, Contact Form 7 Keep in WP Contact and lead forms remain in WordPress.
Security Wordfence, Sucuri, iThemes Security Keep in WP Continue protecting WordPress frontend. BigCommerce handles commerce security.
Caching WP Rocket, W3 Total Cache Keep in WP WordPress caching unaffected. Product data cached by BC Bridge plugin.
Analytics MonsterInsights, Google Site Kit Keep in WP Content analytics in WordPress. E-commerce tracking via BC native integration.

Source: bc-migration architecture decision (ADR-001: Keep WordPress)

Migration Complexity Summary

Based on the plugin mappings, most common WooCommerce plugins have direct equivalents in BigCommerce.

Migration Type Plugin Categories Complexity
Native Payment gateways, channel managers, shipping, wishlists, metafields, multi-storefront Low - Configuration only, no development
App Email marketing, advanced shipping, subscriptions, memberships Low-Medium - App installation and configuration
Keep in WP SEO, page builders, forms, security, caching, content analytics None - No migration required
Custom Highly customized WooCommerce extensions High - Requires development assessment
