BC Bridge Plugin

WordPress plugin that enables existing WordPress/WooCommerce sites to use BigCommerce as their commerce backend.

Overview

BC Bridge is a headless commerce bridge that:

  • Intercepts WooCommerce routes (/shop, /product/*, /cart, /checkout)
  • Fetches data from BigCommerce APIs in real-time (no sync)
  • Renders commerce UI within existing WordPress themes
  • Handles cart state, checkout flow, and customer accounts via BC APIs

Key Principles

  • Stateless First - Minimize WordPress database usage
  • Defensive Initialization - Never crash WordPress on activation
  • Clean Uninstall - Leave no trace when removed
  • API V3 Only - Use BigCommerce V3 APIs exclusively
  • CDN Direct - Use BigCommerce CDN for all assets
